this

	1. 생성된 객체(인스턴스) 자신의 주소(참조, 레퍼런스)이다.
	2. 생성된 객체(인스턴스)가 자신을 부르는 방식이다.
	3. 클래스 구성 요소에 this 활용할 수 있다.
		1) 필드 : this.필드
		2) 생성자 : this()
	4. 주요 활용
		- 필드와 매개변수를 구분하는 용도로 사용한다.
	
		class Person {
			
			// Field
			String name;
		
			// Constructor
			Person(String name) {
				this.name = name;
				// this.name : Field 인 name 을 의미
				// name : 매개변수 name 을 의미
			}
		
		}
